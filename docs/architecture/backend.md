# Tecnologias Utilizadas

Dentro das tecnologias utilizadas, temos que a principal linguagem a ser utilizada é a python, pelo fato de termos utilizado o framework 
Django como principal estrutura do backend, a linguagem a qual é usada dentro do seu desenvolvimento é essa. Essa escolha foi feita 
pensando em alguns pontos primordiais, sendo eles:

- Escalabilidade do Back-end
- Ecossitema de Ferramentas de Tratamento e Manipulação de Dados
- Velocidade e Ergonomia de Desenvolvimento

Com isso, temos que algumas tecnologias que podemos escolher, suprem essas necessidades tanto de forma direta como de forma indireta. E com isso, temos que algumas das decisões tomadas em relação a isso são:

- ***Pandas:*** Trabalhar com dataframes grandes e volumosos, com grande diversidade de dados
- ***Numpy:*** Trabalhar com grandes quantidades de valores que podem ser calculados e usar métodos numéricos
- ***Folium:*** Mapas e visualizações geoespaciais 
- ***Geopandas:*** Trabalhar com dataframes igual ao pandas, com a diferenciação que podem usar dados de geolocalização, de forma que facilite a manipulação de dados geoespaciais
- ***PostgreSQL:*** Banco de Dados relacional robusto, usado para fazer o armazenamento persistente dos dados
- ***PosteGIS:*** Extensão do PostgreSQL para trabalhar, armazenar, e usar dados geoespaciais dentro do nosso sistema
